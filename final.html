Setup

  1. In this working directory, set up the necessary libraries: 
    npm init -y
    npm install --save express express-handlebars express-session sqlite3 sequelize
  2. Start the provided server file: 
    node server.js
  3. Browse to http://localhost:3002/ and you should get the default route.
